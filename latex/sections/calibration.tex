% Section 5: Empirical Results - Calibration Analysis

A critical question for any self-assessment framework is \textbf{calibration accuracy}: do AI-reported epistemic vectors correspond to actual task outcomes? We analyzed calibration using Bayesian belief updating across 82,380 evidence observations.

\subsection{Methodology}

For each vector, we maintain a Bayesian belief about the AI's epistemic state, updated via conjugate normal-normal inference. The prior ($\mu_0$) is set from PREFLIGHT self-assessment; observations accumulate through task execution; the posterior reflects the updated belief at POSTFLIGHT.

The \textbf{learning delta} for each vector is:
\begin{equation}
\Delta_i = \mu_{\text{posterior}} - \mu_{\text{prior}}
\end{equation}

Positive delta indicates the vector value \emph{increased} during task execution (confidence growth). For UNCERTAINTY, which operates on an inverted scale, a \emph{negative} delta (uncertainty decreasing) also indicates confidence growth.

\subsection{The Unified Confidence Pattern}

A critical interpretive point: the data shows \textbf{unified confidence growth}, not asymmetry.

The UNCERTAINTY vector operates on an inverted scale relative to capability vectors:
\begin{itemize}
    \item Capability vectors: Higher value = more capability = more confidence
    \item Uncertainty: \emph{Lower} value = less doubt = more confidence
\end{itemize}

\subsection{Pattern Analysis}

Two patterns emerge from the analysis:

\paragraph{Pattern 1: Learning Produces Confidence Growth}

All vectors show movement toward higher confidence during task execution. AI systems begin tasks with conservative self-assessments and converge toward more accurate (higher) assessments as they accumulate evidence through work.

\paragraph{Pattern 2: Magnitude Varies by Vector}

COMPLETION shows the largest delta (+0.397), suggesting AI systems dramatically underestimate task progress at PREFLIGHT. ENGAGEMENT shows near-zero delta (+0.004), suggesting this vector is already well-calibrated from task onset.

\subsection{Calibration Correction}

The learning delta reveals how self-assessments should be corrected. For PREFLIGHT assessments, adding the expected delta yields a more accurate estimate:

\begin{equation}
V_{\text{corrected}} = V_{\text{PREFLIGHT}} + \Delta_V
\end{equation}

For the Sentinel readiness gate (\Cref{sec:sentinel}), this means:
\begin{itemize}
    \item Raw PREFLIGHT know of 0.55 $\rightarrow$ corrected to $\sim$0.70
    \item Raw PREFLIGHT uncertainty of 0.48 $\rightarrow$ corrected to $\sim$0.33
\end{itemize}

The correction compensates for systematic conservatism at task onset.

\subsection{Limitations}

Several limitations constrain interpretation:

\begin{enumerate}
    \item \textbf{Observational design}: Data derives from naturally-occurring sessions, not controlled experiments with external ground truth.

    \item \textbf{Self-referential measurement}: POSTFLIGHT assessments are also self-reported, not externally validated. The delta measures self-consistency, not necessarily accuracy.

    \item \textbf{Model concentration}: 55\% of observations derive from Claude (Anthropic). Cross-model replication is needed.

    \item \textbf{Temporal confounding}: Data spans framework development. Patterns may reflect evolution rather than stable AI properties.
\end{enumerate}

\subsection{Implications}

The unified confidence growth pattern---all 13 vectors moving toward higher confidence during learning---has several interpretations:

\begin{enumerate}
    \item \textbf{Functional validity}: Self-assessment tracks actual epistemic state, with conservative priors corrected through evidence accumulation.

    \item \textbf{Training-induced conservatism}: RLHF and similar techniques may induce systematic underconfidence, which task execution naturally corrects.

    \item \textbf{Learning signature}: The pattern may simply reflect that doing work increases confidence---a tautology of task completion.
\end{enumerate}

The present contribution is empirical: \textbf{systematic confidence growth during learning is measurable and consistent}. The correction mechanism provides practical value regardless of causal interpretation.
