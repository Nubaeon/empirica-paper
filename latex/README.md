## Compilation

### Local (requires TeX Live or MiKTeX)

```bash
cd latex
pdflatex paper.tex
bibtex paper
pdflatex paper.tex
pdflatex paper.tex
```

Or with latexmk:
```bash
latexmk -pdf paper.tex
```

### Overleaf

1. Create new project on [Overleaf](https://www.overleaf.com)
2. Upload all files from this directory
3. Copy figures from `../figures/` to the project
4. Compile

### Install LaTeX (Ubuntu/Debian)

```bash
sudo apt install texlive-latex-extra texlive-fonts-recommended texlive-science
```

## arXiv Submission

For arXiv, you'll need:
1. Flatten all \input{} commands (or upload all .tex files)
2. Include .bbl file (generated by bibtex)
3. Include figures as PDF/PNG
4. Ensure no external URLs in figure paths
